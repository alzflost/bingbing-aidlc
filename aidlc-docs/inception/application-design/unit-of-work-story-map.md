## Critical Path

```
U1 (Terraform) ──────────────────────────────────────→ 배포 완료
U2 (API: Speaker Mapping) ──→ U2+U3 통합 ──→ E2E
U3 (Agent: Policy + Memory) ─┘
U5 (Frontend: WebSocket) ────────────────────→ U5+U2 통합
U4 (Lambda: Reflection) ────────────────────→ 독립 테스트 후 배포
```

P0 스토리 완료 = S1 + S2 + S3 + S4 시나리오 동작 = 핵심 가치 증명
