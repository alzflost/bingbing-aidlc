# Personas

## P1: 아빠 (actor_father)

| 항목 | 내용 |
|---|---|
| 역할 | 주 운전자, 계정 소유자 |
| 연령대 | 30~50대 |
| 권한 | Full (모든 도구 접근 가능) |
| 응답 스타일 | 효율적, 정보 밀도 높음, 옵션 2~3개 제시 |
| 호칭 | 일상적, 짧게 |
| 핵심 니즈 | 빠른 정보 접근, 운전 중 최소 인터랙션으로 목적 달성 |
| 대표 시나리오 | 경로 안내, 차량 제어, 전화 발신 |

## P2: 엄마 (actor_mother)

| 항목 | 내용 |
|---|---|
| 역할 | 조수석, 일정 관리 주체 |
| 연령대 | 30~50대 |
| 권한 | Full (운전 제어 제외) |
| 응답 스타일 | 협업적, 옵션 제시, 가족 일정 연계 |
| 호칭 | 일상적, 친근 |
| 핵심 니즈 | 가족 일정 조율, 식당/장소 추천, 아이 관리 보조 |
| 대표 시나리오 | 식당 추천(비건 선호), 일정 확인, 아이 콘텐츠 관리 |

## P3: 유아 (actor_toddler)

| 항목 | 내용 |
|---|---|
| 역할 | 뒷좌석 탑승자 |
| 연령대 | 3~6세 |
| 권한 | 최소 (엔터테인먼트 일부만, 안전 최우선) |
| 응답 스타일 | 매우 단순, 친근, 격려적, 짧은 문장 |
| 호칭 | 이름 + 애칭 |
| 핵심 니즈 | 동요, 동화, 간단한 퀴즈 |
| 대표 시나리오 | "노래 틀어줘", 차량 제어 시도 → 안전 차단 |
| 가드레일 | 모든 차량 제어 차단, 콘텐츠 연령 필터(3~6세), 볼륨 제한 |

## P4: 청소년 (actor_teen)

| 항목 | 내용 |
|---|---|
| 역할 | 뒷좌석 탑승자 |
| 연령대 | 13~18세 |
| 권한 | 제한 (엔터테인먼트 + 웹검색, 차량 제어/전화 차단) |
| 응답 스타일 | 캐주얼, 자율성 존중, 안전 가드레일 유지 |
| 호칭 | 이름, 반말 OK |
| 핵심 니즈 | 음악 재생, 웹 검색, SNS 관련 정보 |
| 대표 시나리오 | 음악 변경, 검색, 차량 제어 시도 → 거절 |
| 가드레일 | 차량 제어 차단, 성인 콘텐츠 필터, 전화 발신 차단 |

## P5: 성인 자녀 (actor_adult_child)

| 항목 | 내용 |
|---|---|
| 역할 | 뒷좌석 또는 조수석 탑승자 |
| 연령대 | 19세+ |
| 권한 | 준Full (본인 좌석 환경 + 엔터테인먼트 + 웹검색 + 전화, 내비/운전 제어 제외) |
| 응답 스타일 | 일반 성인 수준, 독립적 |
| 호칭 | 이름, 일상적 |
| 핵심 니즈 | 독립적 콘텐츠 소비, 전화, 검색 |
| 대표 시나리오 | 음악, 전화, 좌석 환경 조절 |
| 가드레일 | 내비게이션 목적지 변경 차단, 운전 관련 차량 제어 차단 |

## P6: 어르신 (actor_elder)

| 항목 | 내용 |
|---|---|
| 역할 | 뒷좌석 동승자 (성별 무관) |
| 연령대 | 60대+ |
| 권한 | 제한 (본인 좌석 환경 + 엔터테인먼트 + 웹검색, 전화는 확인 후) |
| 응답 스타일 | 단순, 핵심만, 느린 발화 속도, 중요 동작 시 1회 재확인 |
| 호칭 | 존칭, 정중 |
| 핵심 니즈 | 좌석 환경 조절(에어컨, 시트), 간단한 정보 조회 |
| 대표 시나리오 | "에어컨 줄여줘", 전화 발신 → 재확인 후 실행 |
| 가드레일 | 내비/운전 제어 차단, 전화 발신 시 재확인 |

## P7: 게스트 (actor_guest)

| 항목 | 내용 |
|---|---|
| 역할 | 미등록 화자 |
| 연령대 | 불명 |
| 권한 | 최소 (정보 조회만) |
| 응답 스타일 | 중립, 친절하지만 거리감 유지 |
| 호칭 | 중립 ("손님", "네") |
| 핵심 니즈 | 기본 정보 조회 (날씨, 시간, 도착 예정 등) |
| 대표 시나리오 | 정보 질문 → 응답, 차량 제어 시도 → 차단 |
| 가드레일 | 메모리 미저장, 모든 제어 차단, 정보 조회만 허용 |
