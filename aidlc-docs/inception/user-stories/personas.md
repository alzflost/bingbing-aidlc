# Personas

> 원본 기획서(family-profile-co-pilot-idea.md §4) 기반 5개 페르소나 정의.
> 7시간 타임박스 내 구현 가능한 범위로 한정.

---

## P1: 아빠 (actor_father)

| 항목 | 내용 |
|---|---|
| 역할 | 주 운전자, 계정 소유자 |
| 연령대 | 30~50대 |
| 권한 | Full (모든 도구 접근 가능) |
| 응답 스타일 | 효율적, 정보 밀도 높음, 옵션 2~3개 제시 |
| 호칭 | 일상적, 짧게 |
| 핵심 니즈 | 빠른 정보 접근, 운전 중 최소 인터랙션으로 목적 달성 |
| 대표 시나리오 | 경로 안내, 차량 제어, 전화 발신, 휴게소 루틴 |
| 매핑 전략 | 운전석 마이크 채널 고정 (Primary), 자기소개 바인딩 (Secondary) |

### 권한 상세

| 도구 | 허용 |
|---|---|
| 내비게이션 (목적지 변경) | ✅ |
| 차량제어 (운전 관련) | ✅ |
| 차량제어 (본인 좌석 환경) | ✅ |
| 뮤직 (재생/변경) | ✅ |
| BT전화 발신 | ✅ |
| 웹검색 | ✅ |

## P3: 아이 (actor_child)

| 항목 | 내용 |
|---|---|
| 역할 | 뒷좌석 탑승자 |
| 연령대 | 8세 |
| 권한 | 제한 (콘텐츠/엔터테인먼트만) |
| 응답 스타일 | 친근, 단순, 안전, 격려적 |
| 호칭 | 이름 + 친근체 |
| 핵심 니즈 | 퀴즈, 음악, 간단한 게임, 호기심 해결 |
| 대표 시나리오 | 공룡 퀴즈, 음악 재생, 차량 제어 시도 → 안전 차단 |
| 매핑 전략 | 뒷좌석 마이크 채널 (Primary), 자기소개 바인딩 (Secondary), 어휘/말투 휴리스틱 (Tertiary) |

### 권한 상세

| 도구 | 허용 |
|---|---|
| 내비게이션 (목적지 변경) | ❌ |
| 차량제어 (운전 관련) | ❌ |
| 차량제어 (본인 좌석 환경) | ⚠️ 제한 (고속 주행 중 차단, 정차 시 부모 확인) |
| 뮤직 (재생/변경) | ⚠️ 연령 필터 + 볼륨 안전 한계 |
| BT전화 발신 | ❌ |
| 웹검색 | ⚠️ 콘텐츠 필터 (폭력·성인·공포 차단) |

### 가드레일

- 모든 차량 운전 제어 차단
- 고속도로 주행 중 좌석 환경 제어(창문 등) 차단
- 콘텐츠 연령 필터 적용 (8세 기준)
- 볼륨 안전 한계 적용
- 부적절 요청 시 "재미있는 다른 것" 우회 응답
- 전화 발신 완전 차단

## P4: 할머니 (actor_elder)

| 항목 | 내용 |
|---|---|
| 역할 | 뒷좌석 동승자 |
| 연령대 | 60대+ |
| 권한 | 제한 (본인 좌석 환경만) |
| 응답 스타일 | 단순, 핵심만, 느린 발화 속도, 중요 동작 시 1회 재확인 |
| 호칭 | 존칭, 정중 |
| 핵심 니즈 | 좌석 환경 조절(에어컨, 시트), 간단한 정보 조회 |
| 대표 시나리오 | "에어컨 줄여줘", 전화 발신 → 재확인 후 실행 |
| 매핑 전략 | 뒷좌석 마이크 채널 (Primary), 자기소개 바인딩 (Secondary) |

### 권한 상세

| 도구 | 허용 |
|---|---|
| 내비게이션 (목적지 변경) | ❌ |
| 차량제어 (운전 관련) | ❌ |
| 차량제어 (본인 좌석 환경) | ✅ |
| 뮤직 (재생/변경) | ✅ |
| BT전화 발신 | ⚠️ 확인 후 |
| 웹검색 | ✅ |

### 가드레일

- 내비게이션/운전 제어 차단
- 전화 발신 시 1회 재확인 필수
- 응답 1문장 우선, 추가 정보는 요청 시에만
- 발화 속도 느리게 조절

## P5: 게스트 (actor_guest)

| 항목 | 내용 |
|---|---|
| 역할 | 미등록 화자 |
| 연령대 | 불명 |
| 권한 | 최소 (정보 조회만) |
| 응답 스타일 | 중립, 친절하지만 거리감 유지 |
| 호칭 | 중립 ("손님", "네") |
| 핵심 니즈 | 기본 정보 조회 (날씨, 시간, 도착 예정 등) |
| 대표 시나리오 | 정보 질문 → 응답, 차량 제어 시도 → 차단 |
| 매핑 전략 | 3단계 매핑 모두 실패 시 자동 할당 |

### 권한 상세

| 도구 | 허용 |
|---|---|
| 내비게이션 (목적지 변경) | ❌ |
| 차량제어 (운전 관련) | ❌ |
| 차량제어 (본인 좌석 환경) | ❌ |
| 뮤직 (재생/변경) | ❌ |
| BT전화 발신 | ❌ |
| 웹검색 | ✅ (정보 조회 only) |

### 가드레일

- 메모리 미저장 (트립 종료 시 모든 발화 데이터 삭제)
- 모든 제어 기능 차단
- 정보 조회만 허용
- 개인화 메모리 미적용

## 페르소나 간 우선순위 (동시 발화 시)

| 우선순위 | 조건 |
|---|---|
| 1순위 | 아빠 (운전자) — 안전 관련 요청 시 절대 우선 |
| 2순위 | 아빠 (운전자) — 비안전 요청 시에도 약간 우선 |
| 3순위 | 엄마, 할머니 — 선착순 |
| 4순위 | 아이 — 큐잉 후 순차 처리 |
| 5순위 | 게스트 — 최후순위 |

> 정차 중에는 운전자 특별 우선권 없음, 선착순 처리.
